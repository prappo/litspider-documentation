Overview

LitSpider is social media scraping tool with SaaS features.

Installation

Server Requirements

Before you install LitSpider make sure your server meets the following Requirements

  • PHP 7.1.13+
  • MySQL 5.6+ or MariaDB 10.0+
  • Intl PHP Extension
  • OpenSSL PHP Extension
  • PDO PHP Extension
  • Mbstring PHP Extension
  • Tokenizer PHP Extension
  • XML PHP Extension
  • Ctype PHP Extension
  • JSON PHP Extension

Installation LitSpider

Step 1 , Extract and Upload file

Unzip the downloaded archive package. Rename the LitSpider directory to your desired directory name and upload the directory to your web server through FTP or Control Panel.

Step 2 , Creating Database

Create a database for LitSpider through your server control panel. If your server has phpMyAdmin, then you can also create a database using phpMyAdmin.

Step 3: Run Installation Wizard

Go to your website address, then you'll see an installation wizard. If you don't see then go to /install route and you will see the installation wizard

Package

LitSpider is a SaaS script. So focused on SaaS development. To use each feature user must select / buy a package to use. You can create different type of package for the same feature. If you want to give some feature free to use then you can also do that. Here below we described how to manage packages.

Creating package

You already know to use a feature user must select / buy a package. To create a package to to Package Settings > Add Package . Then you will see the list of features / plugins available in the script. We made the whole system moduler. We called it plugin or you can call it feature. Even you can disable / delete the entire module / plugin / feature if you want. Anyway after going to the Add Package section you will see all the plugin / feature installed.

You can crete package for a plugin by simply clicking Create package button of feature.

Fill the package details with name , description , price and duration. If you want to make a free package then set 0 to the package price

Edit Package

To edit a package go to Package Settings > Packages You can edit / delete the package you have been created.

Admin Settings

We made a complete admin solution for your software. Everything you need as a admin we included it. Here we described admin and web front / site / frontend features

User Profile

User Management

Plugins

Plugins are the main features of the script. As we told you before we made this script moduler for easy to manage features. If you don't want any feature for example you don't want to use "Facebook" feature in your script then simple go to the Admin Settings > Plugins and disable it. You can also enable this feature whenever you want.

Themes

Theme system available for frontend. You can customize your site and use our built themes. We made 3 themes for you but if you want more theme or you want your own customized theme then you can contact the support and if you have any existing html template that you want to use as a theme then you can also do that. Check our theme folder exists in "public" folder where you can find the theme structure and more. To change themes go to Admin settings > Themes

You can also the theme preview. Every theme has theme settings except the blank theme. You can find the theme settings after activating theme on Page Settings > Front Page
